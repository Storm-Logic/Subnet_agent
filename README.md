# Bittensor Subnet Discord Bot

A Discord bot for your Bittensor subnet. Answers questions about live chain data, miner stats, subnet docs, and more.

## Architecture

```
User query
    │
    ├── Chain + wandb + GitHub docs context
    ├── UID query → add specific UID context
    └── Discord answer generated by Codex CLI
```

## Setup

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

Install and authenticate Codex CLI on the same machine that runs the bot:

```bash
codex login
codex exec "Say hello"
```

### 2. Configure environment

```bash
cp .env.example .env
# Fill in your keys in .env
```

### 3. Run the bot

```bash
python -m bot.main
```

### 4. (Optional) Run webhook receiver

```bash
python webhook.py
```
Set up a GitHub webhook pointing to `http://your-server:8080/webhook`.

---

## Discord Chat

Users can @mention the bot, reply to one of its messages, or DM it with any question. The bot will:
- Inject live chain data, wandb data, reference links, manual Q&A, and configured GitHub docs into every Codex CLI request
- Use `CODEX_MODEL` for factual answers and `CODEX_CREATIVE_MODEL` for creative/vision questions when configured
- Browse and analyze HTTP/HTTPS URLs from user questions or reference links when `CODEX_WEB_SEARCH=live`
- Browse every configured reference URL for each answer when `CODEX_BROWSE_ALL_REFERENCE_URLS=true`

No slash commands or text commands are registered.
